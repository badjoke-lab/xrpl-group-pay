# PR #51 changelog fragment

## Added

- Network-scoped XRPL Source Tag resolution.
- Dedicated Mainnet Source Tag build and runtime approval controls.
- Cloudflare binding support for Mainnet Source Tag configuration.

## Changed

- Testnet prefers `XRPL_TESTNET_SOURCE_TAG` while retaining `XRPL_SOURCE_TAG` as a Testnet-only compatibility fallback.
- Mainnet build and runtime target resolution now require `XRPL_MAINNET_SOURCE_TAG` and explicit Source Tag approval.
- The Mainnet Source Tag readiness check is marked complete while the overall Mainnet gate remains blocked.

## Safety

- Mainnet never falls back to Testnet or legacy Source Tag values.
- Missing, malformed, or unapproved Mainnet Source Tags fail closed.
- The committed Mainnet Worker target keeps Source Tag approval disabled by default.
