# PR #46 changelog fragment

## Added

- Server-authoritative Bill input contracts for Equal, Percentage, Shares, and Custom Amount allocation.
- Stable participant IDs for strategy weights and explicit remainder assignment.

- Bill review and publication now recompute participant obligations through the deterministic allocation engine.

- Strategy participant coverage, percentage totals, share weights, positive obligations, remainder assignment, and final Bill totals are validated before persistence.
