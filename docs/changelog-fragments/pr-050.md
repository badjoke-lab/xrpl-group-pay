# PR #50 changelog fragment

## Added

- A machine-readable XRPL Mainnet Settlement Asset registry.
- Exact identity validation for Mainnet XRP and official RLUSD.
- A runtime Mainnet Asset access layer that requires explicit Mainnet gate approval.

## Changed

- Application builds, validation, and Mainnet deployment now verify the Mainnet Asset registry.
- The Mainnet Asset registry readiness check is marked complete while the overall Mainnet gate remains blocked.

## Safety

- Testnet or unknown Asset IDs cannot be selected through the Mainnet registry.
- Mainnet RLUSD issuer, currency code, precision, verification strategy, and receipt contract are checked exactly.
- Passing the Asset identity check alone does not enable Mainnet operation.
